installer wamp server ou xamp server sur votre machine
-----------------------------
Dans phpmyadmin créer une base de données avec le nom "mundiademande" et importer la bdd qui se trouve dans le répertoire  "mundiademande.sql" 
-----------------------------
déplacer le répertoire gd dans C:\xampp\htdocs ou C:\wampp\www si vous utilisez wamp server
-----------------------------
accéder à la page d'acceuil de site avec le lien:  http://localhost/gd/index.php?page=page-login-v2
-----------------------------


----------------------------
une fois connecter vous pouvez créer des demandes et voir leur suivi.
